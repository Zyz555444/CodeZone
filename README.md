# CodeZone

强大的编程团队协作开发平台 🚀

## 快速开始

```bash
# 克隆项目
git clone <repository-url>
cd codezone

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000
