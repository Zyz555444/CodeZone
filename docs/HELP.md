# CodeZone 帮助中心

## 快速入门

### 1. 注册账号

访问 [/register](/register) 页面，填写以下信息：
- 邮箱地址
- 用户名
- 密码（至少 6 位）

### 2. 创建第一个项目

登录后，在仪表板点击"新建项目"按钮：
- 输入项目名称
- 填写项目描述（可选）
- 选择可见性（公开/私有）

### 3. 邀请团队成员

在项目页面，点击"成员"标签：
- 输入成员的邮箱或用户名
- 设置成员角色
- 发送邀请

### 4. 创建任务

在任务页面，点击"新建任务"：
- 填写任务标题和描述
- 设置优先级
- 分配负责人
- 设定截止日期

### 5. 开始协作编码

进入代码编辑器：
- 浏览文件树
- 点击文件开始编辑
- 实时看到队友的光标和修改
- 使用聊天功能讨论

## 使用指南

### 项目管理

#### 创建项目
```
1. 点击"新建项目"
2. 填写项目信息
3. 选择可见性
4. 点击"创建"
```

#### 邀请成员
```
1. 进入项目页面
2. 点击"成员"标签
3. 搜索用户
4. 设置角色并邀请
```

### 任务管理

#### 看板视图
任务看板包含 4 个默认状态列：
- **待办** - 新创建的任务
- **进行中** - 正在处理的任务
- **审查中** - 等待代码审查
- **已完成** - 已完成的任务

拖拽任务卡片可以在不同状态间移动。

#### 创建子任务
```
1. 打开任务详情
2. 在"子任务"部分
3. 输入子任务标题
4. 按 Enter 或点击"+"
```

### 代码编辑器

#### 文件操作
- **新建文件**: 点击文件树旁的"+"按钮
- **重命名**: 右键点击文件，选择"重命名"
- **删除**: 右键点击文件，选择"删除"

#### 协作编辑
- 看到其他用户的彩色光标
- 实时同步代码变更
- 使用右侧聊天讨论代码

#### 支持的编程语言
CodeEditor 支持 100+ 种编程语言，包括：
- TypeScript/JavaScript
- Python
- Java
- C/C++
- Go
- Rust
- PHP
- Ruby
- Swift
- 等等

### 即时通讯

#### 聊天室
- **全局聊天** - 所有用户都可以加入
- **项目聊天** - 项目成员专属
- **私信** - 直接联系特定用户

#### 消息功能
- 发送文本消息
- 发送系统通知
- 查看在线用户
- 查看历史消息

## 常见问题

### 账号相关

**Q: 如何重置密码？**
A: 目前需要联系管理员重置密码。我们正在开发自助重置功能。

**Q: 可以修改用户名吗？**
A: 暂不支持修改用户名，请谨慎设置。

**Q: 如何注销账号？**
A: 联系管理员进行账号注销。

### 项目相关

**Q: 项目可见性能修改吗？**
A: 可以，项目所有者可以在项目设置中修改可见性。

**Q: 如何删除项目？**
A: 只有项目所有者可以删除项目，在项目设置中找到"删除项目"选项。

**Q: 项目成员有哪些角色？**
A: 目前支持以下角色：
- OWNER - 项目所有者
- ADMIN - 管理员
- MANAGER - 经理
- MEMBER - 普通成员
- GUEST - 访客

### 任务相关

**Q: 如何删除任务？**
A: 在任务详情页，目前没有删除按钮（防止误删），请联系管理员。

**Q: 任务可以分配给多人吗？**
A: 当前版本每个任务只能分配给一个负责人。我们可以通过子任务实现多人协作。

**Q: 如何筛选任务？**
A: 使用顶部的搜索框和筛选器：
- 按状态筛选
- 按优先级筛选
- 按负责人筛选
- 关键词搜索

### 代码编辑

**Q: 代码会自动保存吗？**
A: 是的，每次修改都会实时同步到服务器。

**Q: 如何查看修改历史？**
A: 代码历史版本功能开发中，敬请期待。

**Q: 支持多人同时编辑吗？**
A: 支持！多个用户可以同时编辑同一个文件，变更会实时同步。

### 即时通讯

**Q: 聊天记录会保存吗？**
A: 是的，每个房间的最近 200 条消息会被保存。

**Q: 如何退出聊天室？**
A: 关闭页面即可自动退出。

## 快捷键

### 全局快捷键
- `Ctrl/Cmd + K` - 快速搜索
- `Ctrl/Cmd + B` - 切换侧边栏
- `Ctrl/Cmd + ,` - 打开设置

### 编辑器快捷键
- `Ctrl/Cmd + S` - 保存文件
- `Ctrl/Cmd + F` - 查找
- `Ctrl/Cmd + H` - 替换
- `Ctrl/Cmd + /` - 注释/取消注释
- `Tab` - 缩进
- `Shift + Tab` - 取消缩进

### 聊天快捷键
- `Enter` - 发送消息
- `Shift + Enter` - 换行
